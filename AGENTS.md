# Repository Content

This repository contains a curated index of Gazebo documentation and resources, organized by categories.

## Commit Format

When contributing to this repository, please use the following commit message format:

`[$Emoji] $commit_message`

Where `$commit_message` is the title of the new item being added.

### Emoji List

Please use the following emojis corresponding to the category of the item you are adding:

- :inbox_tray: **Installing and Setup Gazebo**
  - :cd: Gazebo, Gazebo-Classic and Ignition
  - :computer: Installing on Linux, MacOSX and Windows
  - :cloud: Gazebo on virtualized environments
  - :penguin: Gazebo + ROS installation on Ubuntu Linux
  - :cityscape: Large environments resources
  - :test_tube: Headless simulations and CI

- :keyboard: **Programming with Gazebo**
  - :hammer_and_wrench: Development environment
  - :electric_plug: Developing Plugins
  - :snake: Python bindings
  - :envelope: Transport and messages
  - :camera: Working with Camera Images

- :turtle: **Gazebo and ROS**
  - :book: Installation & Basics
  - :page_facing_up: Description formats: URDF, SDF and others
  - :bridge_at_night: The ROS 2 Gazebo Bridge (ros_gz)
  - :satellite: Gazebo sensors and plugins in ROS

  - :compass: Gazebo with Nav2

- :package: **Worlds and 3D Models**
